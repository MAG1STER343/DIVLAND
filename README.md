# UNNVERSI / DIEVERSI

Чёрно‑белый дизайн + “стекло/размытие” + глич‑переходы, **псевдо‑3D фон (canvas)** и **профили пользователей**.

## Запуск

### Вариант 1 (только посмотреть дизайн)
- Открой `index.html` в браузере.
- Важно: регистрация/профили/загрузки **не заработают** на `file://` (нужен сервер).

### Вариант 2 (полный функционал: профили, mp3/mp4)
1) Установи **Node.js LTS** (у тебя он ставится через `winget install OpenJS.NodeJS.LTS`).
2) Установи зависимости сервера:

```bash
cd server
npm install
```

3) Установи **ffmpeg** (нужно для удаления звука из mp4):
- через winget: `winget install Gyan.FFmpeg`
- или через Chocolatey: `choco install ffmpeg -y`

4) (Опционально) Создай `server/.env` по `server/.env.example`, если хочешь поменять `PORT` или настройки cookie.

5) Запусти сервер:

```bash
cd server
npm start
```

6) Открой сайт:
- `http://localhost:3000`

## Что внутри

- `index.html`: страницы **Главная / Профиль / DIEVERSI / DISCORD** + публичный профиль `/u/:slug`
- `styles.css`: glassmorphism + blur + глич на hover + стили “дела” и кастомизации
- `script.js`:
  - canvas‑фон (псевдо‑3D wireframe + быстрый repel от курсора)
  - «Открыть дело» с печатью текста
  - регистрация → подтверждение кода → вход
  - загрузка mp3/mp4 (кастомизация)
  - публичная страница профиля `/u/:slug` (видео‑фон + музыка)
- `assets/`: картинки
- `sounds/`: загруженные mp3 (≤ 40MB)
- `videos/`: загруженные mp4 (≤ 50MB, **звук удаляется**)
- `server/`: backend (Express + SQLite через sql.js)

## Примечания

- **Автозвук**: браузеры часто блокируют автопроигрывание — поэтому для музыки есть кнопка **«Включить звук»**.
- **Домен**: чтобы сайт работал как `unnversi.com`, нужен домен + DNS + reverse‑proxy (например Caddy) и проброс портов на роутере.
